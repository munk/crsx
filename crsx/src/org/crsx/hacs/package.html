This package contains the HACS compiler generation frontend for CRSX. The frontend has the following passes:
  1. CookPG: Loads HACS file using the generic Raw1 parser and generates metaparser, an internal .pg parser for the user's terms.
  2. CookLoad: Combines the generated metaparser with the skeleton Raw HACS parser to generate HACS-term version of user's HACS file.
  3. CookCRS: Converts user's HACS-term version of the HACS rules to generic CRSX3 form.
After this the usual CRSX3 compilation process should be used.

Note: Do not store any .java or .jj files in this directory!
